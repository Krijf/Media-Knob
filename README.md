# Media-Knob

Multi purpose Media Knob/Dial for Windows

emulate a usb HID device with the hid project lib.


Features:
- Volume Control
- Mute
- Mikrofon Mute
- 4 extra Buttons
- 3D printed case
- case is created with fusion 360
